📝 Simple HTML Registration Form
A clean, responsive, and user-friendly registration form built with HTML5 and CSS3. This project demonstrates basic form validation, input types, and layout design.

🚀 Features
Responsive Design: Works on desktops, tablets, and mobile phones.

Form Validation: Uses native HTML5 validation (e.g., required, type="email").

Clean UI: Simple styling for a professional look.

User Fields: Includes Name, Email, Password, Gender selection, and Terms of Service checkbox.
